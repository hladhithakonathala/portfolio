## 🚀 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/hladhithakonathala/hladhithakonathala.github.io.git
   ```

2. Open the folder:
   ```bash
   cd hladhithakonathala.github.io
   ```

3. Open `index.html` in your browser — no server needed!

---

## 🌍 How to Deploy on GitHub Pages (Free Hosting)

1. Create a new GitHub repository named exactly:
   ```
   hladhithakonathala.github.io
   ```

2. Upload `index.html` (rename `portfolio.html` → `index.html` before uploading)

3. Go to **Settings → Pages → Source → main branch**

4. Your site will be live at:
   ```
   https://hladhithakonathala.github.io
   ```
